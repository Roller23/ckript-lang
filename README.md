# Ckript language
An interpreter for my own programming language (Ckript)

Cheatsheet:

Built in types

* int
* str
* double
* arr
* obj
* func

Declaring variables

```
int number = 5;
```

Declaring functions

```
type name = function(args...) return_type { function_body };
```

Example:

```
func square = function(int x) int {
  return x * x;
};
```
